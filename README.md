Grammar Guru


Grammar Guru is an AI-powered English grammar learning and writing assistant designed to help users improve their grammar, vocabulary, and writing skills through intelligent feedback and interactive exercises.

Whether you're a student preparing for exams, a professional writing emails, or someone learning English, Grammar Guru makes learning grammar simple and engaging.


##  Features

###  AI Grammar Checker
- Interactive text editor
- Grammar analysis interface
- Grammar score display
- Grammar explanation panel
- Writing suggestions
- Text formatting option


### 📖 Grammar Insights

- Example usage
